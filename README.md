# VIMEOをLaravelで使うサンプル

## 前提条件
- sailを使っています。

## 設定項目
- .envに下記項目を入力
```
VIMEO_CLIENT=
VIMEO_SECRET=
VIMEO_ACCESS=
```

### Vimeoでアカウント作成
[VIMEO](https://vimeo.com/)

### 開発者としてログイン
[Developer](https://developer.vimeo.com/)

### アプリを作成
[create App](https://developer.vimeo.com/apps/new)

### トークン作成
[my App](https://developer.vimeo.com/apps)
- このとき、public以外にprivate、uploadを選択しておきましょう
